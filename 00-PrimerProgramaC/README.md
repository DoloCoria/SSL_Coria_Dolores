### TP0 - Primeros pasos en Git, GitHub, Entorno de Desarrollo

Descripción: Primer trabajo práctico para la instalación y configuración de todo lo necesario para la cursada. En el TP realizamos la configuración del perfil de GitHub, instalación de Git, creación del Repositorio para la cursada, instalación del compilador gcc, y el editor VSC. También creamos nuestro primer programa en C.
